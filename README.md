# AI-Powered Fake Job Detection System

An AI-powered system that detects fraudulent job postings using DistilBERT, NLP and rule-based fraud detection.

## Features

- DistilBERT Deep Learning Model
- Fraud Detection Rules
- Risk Scoring Engine
- Explainable AI Indicators
- Streamlit Dashboard

## Tech Stack

- Python
- PyTorch
- Hugging Face Transformers
- Streamlit
- NLP

## Results

- Accuracy: 98.95%
- F1 Score: 88.15%

## Run Locally

pip install -r requirements.txt

streamlit run app.py
